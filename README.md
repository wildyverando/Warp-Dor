# WarpDor
WarpDor is Tools used to inject unlimited warp quota for free

# Installation
```bash
git clone https://github.com/wildyverando/Warp-Dor.git
cd Warp-Dor

# Copy proxy list and rename it as active.txt then Run
go mod tidy
go run main.go

# Or you can build it
go build ./main.go
```
